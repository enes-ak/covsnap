# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. All contributors, maintainers, issue reporters, and
participants in project spaces (including the GitHub repository, issue tracker,
and pull requests) are expected to follow it.

## Summary

- Be welcoming and respectful to people of all backgrounds and experience levels.
- Assume good intent; disagree on technical merits, not on people.
- Keep discussion focused on the project and its goals.
- Unacceptable behavior includes personal attacks, discriminatory language,
  and sustained disruption of discussion. The full list of unacceptable
  behaviors is defined in the Contributor Covenant linked above.

## Reporting

If you experience or witness behavior that violates this code of conduct,
please report it by opening a confidential contact with the project maintainer
through their GitHub profile at <https://github.com/enes-ak>. Reports will be
handled discreetly. Where possible, please include a description of the
incident, links to any public artifacts (issues, comments, commits), and the
outcome you are seeking.

Do not post reports as public issues.

## Enforcement

The project maintainer is responsible for clarifying and enforcing this code
of conduct and will take appropriate and fair corrective action in response to
any behavior deemed inappropriate, threatening, or harmful. Enforcement
follows the community impact guidelines described in the Contributor Covenant
(correction, warning, temporary ban, permanent ban).
